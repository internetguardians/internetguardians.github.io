how to publish a new article

first: clone the repository

go to the folder where you want to clone the repository, then use git clone to clone it.

Second: copy this template

you need to copy this template then start working. if you want to copy, just go to any old article and copy its content, or click on this button

article title

edit the main page title for your article.

article description

edit the main page description for your article.

the heading

put your heading in the h2 tag, and your image name in the img tag.

write your content

write your content as shown in the image below..

Finally: publish the article

now, your article is ready. first make sure that your files are up-to-date by using the command git pull --rebase. then use the these three commands in order:

  1. git add *
  2. git commit -m "Published a new article"
  3. git push origin main

then wait loading

congratulations!

now your code is ready to be seen by the public!
